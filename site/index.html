Get Started

Up and running in under a minute

# Install PodForge
$ pip install podforge

# Generate a podcast from a topic
$ podforge --topic "The future of AI"

# Generate from a URL
$ podforge --url "https://example.com/article"

# Generate from a PDF
$ podforge --pdf "research-paper.pdf"

# Use free TTS (no API key needed)
$ podforge --topic "Space exploration" --tts free
